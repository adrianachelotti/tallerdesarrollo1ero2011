
Requiere Tomcat y Maven.

Importar en Eclipse el proyecto, a través de File > Import > Existing Projects into workspace.

1) Para activar Maven, botón derecho sobre el proyecto > Maven > Enable dependency management
2) Para hacer el deployment, igual que antes, agregarlo a Tomcat a través de Add and remove. Start/Restart server. En el navegador escribir http://localhost:8080/sve/
